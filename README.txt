
kbtypedef_common
===============================

This module is the KBase repository of common typed object definitions used across the project, and also
includes a java library for manipulating, storing, and validating typed objects.


Below is an index of the contents of each folder:

docs
	documentation of common typed objects and the typed object framework
	
include
	the folder which includes the typedef of the common shared types
	
kb-typed-obj-lib
	the java library for manipulating and validating instances of typed objects

schema
	storage of json schema documents representing common typed objects for convenience, but
	the official up-to-date typed object definitions will be in the 'include' directory
